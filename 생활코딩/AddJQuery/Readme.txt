라이ㅣ브러리 vs 프레임워크
다른사람의 도움을 받아 프로그램을 개발한다.
라이브러리 : 내가 만들고자 하는 프로그램의 부품이 잘 정돈되어있는 재사용되기 쉬운 소프트웨어
내가 만들고있는 프로그램에서 사용하는 부품을 가져오는 느낌

프레임워크: 우리가 만들고자하는것이 어떤것이 있을때 게임, 웹등등이 있을때 만들고자 하는것이 어떤것이냐에 따라서
공통점, 기획의도에따라 달라지는점이 있다.
이 공통점들은 프레임워크가 해결해주고 살짝살짝 바꿔주면 프로그램을 쉽게쉽게 만들수있다.

둘다 다른사람과 협력하는 모델이다.

jQuery 는 라이브러리이다.
content delivery network : 서버에 많은 스크립트들을 가져다놓고 script src를 통해서 우리가 사용할수 있게 해준다.
